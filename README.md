# Gender Pay Gap SQL Analysis

## Overview

This project explores gender pay gap data across companies in the United Kingdom using SQL and data analysis techniques.

## Objectives

* Analyze gender pay gaps across UK companies
* Compare industries and regions
* Study bonus gaps
* Examine trends over time
* Identify companies with the largest pay gaps

## Tools Used

* SQLite
* SQL
* Excel
* Data Analysis
* Data Visualization

## Key Analysis

* Data cleaning and validation
* Missing values analysis
* Industry comparison
* Regional comparison
* Bonus gap analysis
* Yearly trend analysis

## Project Files

* SQLite database
* SQL queries
* Analysis presentation
